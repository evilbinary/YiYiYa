# 文档组织规范

## 目录结构原则

### 1. 按功能分类

```
docs/
├── install/      # 安装相关文档
├── develop/      # 开发指南文档
├── architecture/ # 架构设计文档
├── api/          # API 参考文档
├── material/     # 参考资料（芯片手册、协议等）
├── demo/         # 演示材料（截图、视频）
└── image/        # 文档使用的图片资源
```

### 2. 文档分类

#### 安装文档 (`install/`)
- 平台安装指南
- 环境配置
- 工具安装

#### 开发文档 (`develop/`)
- 入门教程
- 开发指南
- 最佳实践
- 子目录:
  - `driver/` - 驱动详细文档
  - `image/` - 文档图片

#### 架构文档 (`architecture/`)
- 系统架构设计
- 设计决策
- 架构图

#### API 文档 (`api/`)
- 系统调用
- 内核 API
- 驱动接口

#### 参考资料 (`material/`)
- 硬件手册摘要
- 协议说明
- 算法原理

## 命名规范

### 文件命名
- 使用有意义的英文名称
- 多个单词用连字符 `-` 连接
- 示例: `bcm2837-network.md`, `driver-development.md`

### 中文文档
- 开发指南类文档可使用中文命名
- 示例: `内核开发.md`, `驱动开发.md`
- API 文档推荐使用英文命名

## 文档内容要求

### 1. 标题结构
```markdown
# 主标题

## 二级标题

### 三级标题
```

### 2. 必需元素
- **概述**: 简要说明文档内容
- **目录**: 长文档需要目录
- **示例代码**: 包含可运行的代码示例
- **参考资料**: 相关链接和文档

### 3. 代码示例
使用代码块并指定语言：
````markdown
```c
int main() {
    return 0;
}
```
````

### 4. 链接引用
使用相对路径引用其他文档：
```markdown
详见 [驱动开发](../develop/驱动开发.md)
```

## 图片管理

### 图片位置
- 文档特定图片: `docs/develop/image/`
- 通用图片资源: `docs/image/`

### 图片引用
```markdown
![架构图](../image/arch.png)
```

### 图片命名
- 使用有意义的名称
- 包含日期或版本信息（可选）
- 示例: `bcm2837-usb-arch.png`

## 文档索引

### README.md
每个目录应有 `README.md` 作为索引：
- 目录用途说明
- 文档列表
- 快速导航

### 主索引
`docs/README.md` 作为总索引：
- 文档导航
- 快速开始
- 目录结构说明

## 文档更新流程

1. **新增文档**
   - 确定分类目录
   - 按命名规范创建文件
   - 更新相应的 README.md

2. **修改文档**
   - 更新内容
   - 更新修改日期（可选）
   - 检查相关链接

3. **移动文档**
   - 更新所有引用链接
   - 更新 README.md
   - 通知相关开发者

## 文档模板

### 驱动文档模板
```markdown
# [驱动名称] 驱动开发文档

## 概述
简要说明驱动功能和用途。

## 硬件架构
硬件连接和架构图。

## 文件结构
相关源码文件列表。

## 初始化流程
驱动的初始化过程。

## API 接口
导出的接口函数说明。

## 使用示例
实际使用的代码示例。

## 当前实现状态
- [x] 已完成功能
- [ ] 待实现功能

## 参考资料
相关硬件手册和文档链接。
```

### API 文档模板
```markdown
# [模块名称] API 参考

## 概述
模块功能和用途。

## 函数列表
| 函数 | 说明 | 状态 |
|------|------|------|
| func1() | 功能说明 | ✅ |

## 详细说明

### func1()
**功能**: 函数功能说明

**原型**:
```c
int func1(int param);
```

**参数**:
- `param`: 参数说明

**返回值**: 返回值说明

**示例**:
```c
int ret = func1(10);
```

## 参考资料
- 相关文档链接
```
